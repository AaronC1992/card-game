# Filthy Minded Battle Deck

A neon-dark browser card game with creatures, actions, items, reactions, and field cards. Battle your way through story mode or jump straight in with a custom deck.

[![Play Now](https://img.shields.io/badge/Play%20Now-%E2%96%B6%20Launch%20Game-ff2d78?style=for-the-badge&labelColor=0a0a0f)](https://aaronc1992.github.io/card-game/)

</div>

---

## Features

- **Story Mode** — start with a starter pack and unlock more cards as you beat chapters
- **Quick Play** — build a custom deck and battle instantly
- **Deck Builder** — mix creatures, actions, items, reactions, and field cards
- **Bot AI** — automated opponent with adaptive card play
- **Card Art** — AI generated art for every card

## Tech Stack

- React + TypeScript
- Vite
- CSS Modules

## Run Locally

```bash
npm install
npm run dev
```
